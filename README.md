## HOW TO USE

<b>Prerequisite: you must know how to get the content query selector to retrieve the content of novel.</b>

### 1. Install NodeJS
Install NodeJS from https://nodejs.org/en, install tutorial can be found on Youtube.
### 2. Run the tool
1. You need to open this on the terminal. Make sure the directory is correct.
2. Run the following command
```
npm i
```
3. Add your browserless API key in .env file, next to BROWSERLESS_TOKEN
4. Run the following commands
```
npm run build
npm start
```
5. Open localhost:3000
